## Configuration

### Virtualenv

Create

```
virtualenv env
```

Active

#### Linux

```
source env/bin/activate
```

#### Windows

```
env\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

## Run

```
python bot.py
```

## (Option) Docker

```
docker-compose up
```
